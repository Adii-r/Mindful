# 🌿 Mindful

# 📖 Project Overview

Mindful aims to encourage positive habits and mental wellness by providing users with personalized assistant, inspirational quotes, books, daily recommendations, and interactive games while allowing administrators to manage all application content through an intuitive web dashboard.
